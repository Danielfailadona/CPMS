Interactive Button Highlight

Click any button to highlight it in orange. The highlight will move to the most recently clicked button.

Select an Option

How It Works

This implementation uses JavaScript to add an "active" class to the clicked button, which applies the orange background color.

When another button is clicked, the "active" class is removed from all buttons and then applied to the newly clicked button.

The transition is smooth thanks to CSS transitions, creating a pleasant user experience.
